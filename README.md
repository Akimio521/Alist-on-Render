# Alist-on-Render
- 在Render上部署Alst
- Alist版本v3
## 一键部署
[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)
### 原版
Branch（分支）选择main
### PostgreSQL数据库
Branch（分支）选择PostgreSQL
### MySQL数据库
Branch（分支）选择MySQL
